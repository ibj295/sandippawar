### About

I work end-to-end across the Microsoft Fabric and Power BI stack — from ingesting raw source data through Data Factory, cleaning and modeling it with PySpark (including building bridge/dimension tables out of duplicate-heavy sources), to shipping fast, governed Power BI Direct Lake reports.

I also write about the stack — breaking down Fabric and Power BI concepts (Direct Lake vs. Import vs. DirectQuery, Lakehouse vs. Warehouse, pipeline connection types) for other analysts.
